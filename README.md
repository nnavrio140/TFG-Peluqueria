# Peluqueria Almáchar
## Tematica
 - Peluquería 
## Objetivos
 - Es una aplicacion que sirve para gestionar las citas de una peluqueria
## Funcionalidades
- Roles
  -- Admin: Podra ver todas las citas gestio
  -- Cliente: Podra ver sus citas
- Vistas:
  -- Home
  -- Gestion de citas: lleva un historico y creacion de vista
  -- Login
  -- Register
  -- Nosotros
  -- Contacto
  -- CRUD usuario y citas
  -- CRUD Perfil
## Arquitectura/Tecnología
 - Frontend:
   -- React
   -- Module CSS
   -- Librerias
 - Backend:
   -- Laravel
   -- ORM: Eloquent
   -- Base de datos: MySql
 - Despliegue:
   -- Docker
   -- AWS Amazon
